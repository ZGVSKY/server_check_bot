# PC Control Telegram Bot

Телеграм-бот на базі **aiogram v3** для дистанційного керування комп'ютером через API.

## 🚀 Функціонал
- **Вимкнення та Перезавантаження ПК**: Дистанційне керування живленням.
- **Статистика**: Отримання показників системи (CPU, RAM, Temp).
- **Скріншот**: Можливість отримати поточне зображення екрана.
- **Керування звуком**: Регулювання гучності та режим "Без звуку".
- **Безпека**: Доступ лише для адміністратора.

## 🏗 Архітектура
Проект побудований за модульним принципом:
- `handlers/`: Логіка обробки повідомлень та команд.
- `keyboards/`: Скрипти для генерації Reply та Inline клавіатур.
- `middlewares/`: Проміжне ПЗ (наприклад, логування).
- `filters/`: Кастомні фільтри (перевірка на права адміністратора).
- `utils/`: Допоміжні сервіси, зокрема `pc_api.py` для зв'язку з API комп'ютера.
- `config.py`: Керування налаштуваннями через Pydantic.

## 🛠 Налаштування

1. **Клонуйте репозиторій**:
   ```bash
   git clone <url_репозиторія>
   cd telegram_bot_pf
   ```

2. **Встановіть залежності**:
   ```bash
   pip install aiogram pydantic-settings python-dotenv aiohttp
   ```

3. **Налаштуйте середовище**:
   Створіть файл `.env` у корені проекту за прикладом:
   ```env
   BOT_TOKEN=ваш_токен
   ADMIN_ID=ваш_telegram_id
   ```

4. **Запустіть бота**:
   ```bash
   python main.py
   ```
