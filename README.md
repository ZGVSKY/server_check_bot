# PC Control Telegram Bot 🛡️

Телеграм-бот на базі **aiogram v3** для дистанційного керування комп'ютером та моніторингу його стану.

## 🚀 Основний функціонал
- **Керування живленням**: Дистанційне вимкнення та перезавантаження ПК.
- **Моніторинг**: Отримання візуальної статистики (CPU, RAM, Temp) у вигляді картинки.
- **Швидкий запуск**: Додавання та запуск будь-яких програм/скриптів одним натисканням.
- **Командний рядок**: Повноцінне виконання команд у терміналі ПК через чат.
- **Медіа**: Зміна гучності (більше/менше/вимкнути) та створення скріншотів екрана.
- **Безпека**: Багаторівнева система доступу (Суперкористувач, Повний доступ, Тільки статистика).
- **Сповіщення**: Автоматичне попередження адміністратора про перегрів (поріг налаштовується).

## 🏗 Архітектура
Проект модульний:
- `handlers/`: Логіка команд та авторизації.
- `keyboards/`: Клавіатури (Reply та Inline).
- `utils/`: API клієнт, генерація зображень та фонові завдання моніторингу.
- `database/`: Локальні JSON-бази користувачів та програм.

## 🛠 Налаштування та запуск

1. **Встановіть залежності**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Налаштуйте `.env`**:
   Створіть файл `.env` у корені:
   ```env
   BOT_TOKEN=ваш_токен_від_BotFather
   ADMIN_ID=ваш_id_телеграм (цифрами)
   TEMP_THRESHOLD=80
   ```

3. **Запуск**:
   Використовуйте `run_bot.bat` (для Windows) або:
   ```bash
   python main.py
   ```

## 📖 Інструкція з використання

### Перший запуск
Коли ви вперше пишете боту (`/start`), він розпізнає вас як Суперкористувача (якщо ваш ID співпадає з `ADMIN_ID`). Вам відкриється повне меню.

### Надання доступу іншим
Коли інший користувач напише боту, ви отримаєте повідомлення з кнопками. Виберіть рівень доступу:
- **Заборонити**: Користувач нічого не зможе робити.
- **Тільки статистика**: Бачить лише стан ПК та список програм.
- **Повний доступ**: Може керувати всім (крім керування іншими користувачами).

### Додавання програм
Натисніть "Додати в швидкий запуск" -> введіть шлях до `.exe` -> введіть назву (або `+` для автоматичної назви). Тепер програма з'явиться в меню "Запуск програм".

### Командний рядок
Натисніть "Командний рядок" -> введіть будь-яку команду (напр. `dir` або `shutdown -a`) -> отримайте результат.
